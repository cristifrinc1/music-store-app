# Music Store Application

A full-stack e-commerce application for musical instruments developed as my bachelor's degree project.

## Features

- User authentication
- Role-based access (Admin/User)
- Product catalog
- Shopping cart
- Order management
- Product reviews
- Admin dashboard

## Technologies

### Backend
- Java
- Spring Boot
- Maven

### Frontend
- Angular
- TypeScript

### Database
- MySQL

## Installation

1. Clone the repository
2. Configure MySQL
3. Run the Spring Boot application
4. Start the Angular frontend
